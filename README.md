# Bot-penasihat-karir-yang-menyarankan-karir-dan-jalur-profesi

Bot ini membantu para remaja atau orang dewasa untuk mencari informasi karir dan juga lowongan kerja. Dengan memberitahu bot apa keahlian dia

## Alat yang akan saya gunakan untuk mengerjakan ide:
* database SQL
  * tabel pertanyaan
  * tabel daftar karir
  * tabel loker
* ui Button dari Discord

## Proyek ini akan berisi:
* `!start` --> menyapa user dan memberikan daftar perintah yang bisa digunakan
* `!tanyakarir <minat>` --> menampilkan saran karir terkait minat user
* `!loker` --> menampil loker yg tersedia
